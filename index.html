Robotframework-debugui

A simple to use debug tool for robotFramework scripting. Allows user to interactively try script commands.

View on GitHub Download .zip Download .tar.gz

Welcome to robotFramework-DebugUI.

This is a quick and simple way to debug and develop robotFramework scripts.

Rapid debugging

Add the word 'Debug' in your script and the debugger panel pops up. This lists available commands and lets you quickly try them interactively.

Simple to use

Add this line to your script settings: 

*** Settings ***
Library   DebugUI

Insert the word "Debug" before the line you want to test. Run the script and the debugger panel pops up. It lists available commands and lets you try them interactively. A simple example :

*** Test Cases ***
Test DebugUI
    Open browser    http://www.ucas.ac.uk    Chrome
    debug

When you are happy with the commands simply paste them into your script. You can use the debugger to develop new scripts or rapidly fix breakages in existing scripts. You can also view robotFramework variables and use other tools like firebug and checkXpath while debugging.

Requirements

  • Python
  • Robotframework
  • Ride
  • Should work with Jython / IronPython

Install

Hopefully a pip install soon. Currently just download the code folder and put it on your PYTHONPATH and good luck !!
